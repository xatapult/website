# `website` [PRIVATE]

This repository holds the source code for the Xatapult websites ([http://www.xatapult.nl](http://www.xatapult.nl) and 
[http://www.xatapult.com](http://www.xatapult.com)), generated with the use of the `xtpxlib-xwebgen` component.

## Usage

To create the website:

- Run `xpl/create-xatapult-website.xpl` (inside oXygen or on the command line). The results can be found in `/build` 
- Move the PRD versions of the sites to the Xatapult production server in `/home/erik/web`
